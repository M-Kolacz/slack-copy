Threaddy – Privacy Policy

Effective date: March 15, 2026

Threaddy is a Chrome extension that copies Slack thread content to your clipboard with one click.

Data collection

Threaddy collects no user data of any kind.

Network requests

Threaddy makes no network requests. It operates entirely within your browser and never contacts any external server.

Storage

Threaddy writes nothing to storage and transmits no data externally.

DOM access

Threaddy only reads already-rendered DOM content within your active Slack session (app.slack.com) in order to extract thread messages for clipboard copying. This content never leaves your browser.
