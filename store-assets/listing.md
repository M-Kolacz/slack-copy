# Chrome Web Store Listing — Threaddy

## Store Title (30 / 45 chars max)

Threaddy – Copy Slack Threads

## Short Description (124 / 132 chars max)

Copy any Slack thread to your clipboard in one click. Clean, markdown-ready output — perfect for AI tools, docs, and notes.

## Long Description

Threaddy adds a small copy button directly inside Slack's thread panel header.

One click captures every message in the open thread and copies it to your clipboard as clean, readable markdown — no setup, no popups, no configuration.

HOW IT WORKS
• Open any thread in Slack (app.slack.com)
• Click the copy icon that appears in the thread panel header
• A checkmark confirms the copy succeeded
• Paste anywhere: Claude, ChatGPT, Notion, Obsidian, Jira, Google Docs

OUTPUT FORMAT
Messages are formatted as:
  **@username** (YYYY-MM-DD HH:MM):
  Message content here.

PRIVACY
Threaddy collects zero data. No network requests. No storage writes. It only reads already-rendered DOM content within your active Slack session. Everything stays in your browser.
Full privacy policy: https://m-kolacz.github.io/slack-copy/

PERFECT FOR
• Developers feeding thread context into AI coding agents
• Product managers capturing decision threads for docs
• Support teams archiving customer conversations
• Anyone who moves Slack context into external tools
